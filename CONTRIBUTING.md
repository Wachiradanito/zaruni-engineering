# Contributing

This repository contains documentation only — no executable code is accepted from external contributors.

If you notice an error or have a suggestion for a case study or architecture document, please open an issue.

## Issues Welcome

- Corrections to technical documentation
- Suggestions for new case studies
- Questions about architecture decisions
- Grammar and clarity improvements

## Not Accepting

- Code contributions (this is a documentation repo)
- Business logic implementations
- Feature requests for the private Zaruni codebase

Thank you for your interest in improving this documentation!
